# E-commerce Product Analysis

This project analyzes mobile phone data (price, ratings, reviews) to identify:

- Value-for-money products  
- Hidden winners  
- Pricing opportunities  

## Tech Used
- Python  
- BeautifulSoup  
- Pandas  
- MySQL  

## Key Insights
- Lower-priced products can outperform expensive ones in ratings  
- Relative review performance helps identify strong demand  
- Market segmentation improves decision-making  

## Output
- Cleaned dataset (CSV)
- Insights for business decisions
